浏览器兼容性问题

发表于 2016-11-20 | 分类于 css
| 字数统计: 1.1k 字

很多浏览器兼容性问题都是因为不同的浏览器对同一段代码有不同的解析，造成页面显示效果不统一的情况。在大多数情况下，我们的需求是，无论用户用什么浏览器来查看我们的网站或者登陆我们的系统，都应该是统一的显示效果。所以浏览器的兼容性问题是前端开发人员经常会碰到和必须要解决的问题。
